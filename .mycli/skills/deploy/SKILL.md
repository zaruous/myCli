$ARGUMENTS 환경으로 배포를 준비합니다.

## 배포 전 확인사항

1. **테스트 확인**
   - 모든 테스트가 통과했는지 확인하세요.

2. **환경 변수 확인**
   - .env 파일이 올바르게 설정되어 있는지 확인하세요.

3. **배포 실행**
   - `npm run build` 로 빌드하세요.
   - 빌드 결과물을 배포 대상으로 전송하세요.

4. **배포 후 검증**
   - 서비스가 정상 동작하는지 확인하세요.
   - 로그에 에러가 없는지 확인하세요.

사용자 확인 없이 배포를 진행하지 마세요.
