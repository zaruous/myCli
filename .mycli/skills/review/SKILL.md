$ARGUMENTS 파일을 읽고 다음 항목을 검토해주세요:

## 검토 항목
1. **코드 품질**: 함수 분리, 변수 명명, 중복 코드
2. **보안**: 입력 검증, 민감 정보 노출, 주입 취약점
3. **성능**: 불필요한 루프, 메모리 누수, 비효율적 처리
4. **가독성**: 주석, 코드 구조, 이해하기 어려운 부분

각 항목별로 발견된 문제와 개선 제안을 한국어로 작성해주세요.
파일 경로가 없으면 현재 디렉터리의 index.js를 검토하세요.
