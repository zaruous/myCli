# mycli 프로젝트 컨텍스트

## 프로젝트 개요
KYJ CLI - Node.js 기반 AI 개발자 도구. LangChain + Gemini를 사용한 대화형 에이전트.

## 기술 스택
- Runtime: Node.js ESM
- LLM: Google Gemini 2.5-flash (기본), OpenAI GPT-4o, Ollama 지원
- Framework: LangChain (@langchain/classic, @langchain/core)
- CLI: Commander.js + Inquirer.js

## 작업 규칙
- 한국어로 응답한다
- 파일 수정 시 반드시 기존 내용을 먼저 읽는다
- 코드 작성 시 ESM(import/export) 문법을 사용한다
- Windows + PowerShell 환경임을 고려한다
